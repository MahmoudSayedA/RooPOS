# RooPOS - Backend

This is the backend API for **RooPOS**, built with **.NET** and designed to work primarily in **offline / local environments**.

---

## 🛠 Technology Stack

- .NET (Web API)
- Entity Framework Core
- SQL Server Express
- RESTful API
- JWT Authentication

---

## Architecture

The backend follows a **modular architecture** with clear separation of concerns:

- Domain
- Application
- Infrastructure
- API

---

## 🔐 Authentication & Authorization

- JWT-based authentication
- Permission-based authorization (not role-hardcoded)
- Users can have **multiple permissions**
- Permissions define access to features

---

## API Style

- RESTful endpoints
- JSON payloads
- Designed for Flutter desktop consumption, and web applications
